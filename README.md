# 赛题任务
初赛赛题任务是对西安的热门景点、美食、特产、民俗、工艺品等图片进行分类，即首先识别出图片中物品的类别（比如大雁塔、肉夹馍等），然后根据图片分类的规则，输出该图片中物品属于景点、美食、特产、民俗和工艺品中的哪一种。共有54个类别。

# 输入输出格式

![alt 仿唐三彩](https://github.com/ielym/Huawei-2019-AI-Competition/blob/main/imgs/img_12.jpg)

  " result ": "工艺品/仿唐三彩"
